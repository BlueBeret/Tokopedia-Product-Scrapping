# Tokopedia-Product-Scrapping
This project was made for Information Retrieval assignment.

## Getting Started
There are 2 script: 
 - `get_tokopedia_links.py`  
 This will scrap all the product links from the recommendation section in tokopedia landing page. Just start the script and ctrl+c to stop and save.
 - `get_tokopedia_details.py`  
 The links from the first script will be the input for this script.
